# Calculadora

Proyecto simple de calculadora en Python con interfaz web y pruebas.

Contenido:
- `Calculadora.py` - código principal
- `calculadora_web.html` - interfaz web
- `tests/` - pruebas unitarias y de Playwright

Instalación y uso

1. Crear un entorno virtual (recomendado):

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

2. Ejecutar la calculadora:

```bash
python Calculadora.py
```

3. Ejecutar pruebas unitarias:

```bash
pip install -r requirements.txt  # si tienes un requirements
pytest
```
